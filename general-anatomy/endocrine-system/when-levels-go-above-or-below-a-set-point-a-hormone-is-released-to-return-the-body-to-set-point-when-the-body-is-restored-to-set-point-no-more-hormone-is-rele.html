When levels go above or below a set point a hormone is released to return the body to set point. When the body is restored to set point no more hormone is released. This is an example of ____

(A) a positive feedback loop

(B) ** a negative feedback loop

(C) a neutral feedback loop

EXPLANATIONS BELOW

Concept note-1: -Glucagon also stimulates adipose cells to release fatty acids into the blood. These actions mediated by glucagon result in an increase in blood glucose levels to normal homeostatic levels.

Concept note-2: -If heat loss is severe, the brain triggers an increase in random signals to skeletal muscles, causing them to contract and producing shivering. The muscle contractions of shivering release heat while using up ATP.

Concept note-3: -The effector is an organ, gland, muscle, or other structure that acts on the signal from the control center to move the variable back toward the set point.

Concept note-4: -This is called a process of alteration of the set point in a feedback loop. Changes can be made in a group of body organ systems in order to maintain a set point in another system. This is called acclimatization.
